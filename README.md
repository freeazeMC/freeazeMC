NodeGuard Browser
NodeGuard, gizlilik odaklı, sıfır log prensibiyle çalışan bir web tarayıcısıdır.
Özellikler:
Sıfır Kayıt: Geçmiş, çerez veya önbellek (cache) diske yazılmaz.
Anonimlik: Oturum bazlı, RAM üzerinde çalışır.
Minimalist Tasarım: Gereksiz scriptler ve izleyicilerden arındırılmıştır.
Nasıl Çalıştırılır?
python browser.py komutu ile tarayıcıyı başlatabilirsiniz.
